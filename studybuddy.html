Project Information

With dozens of fun commands, Study Buddy motivates the user and helps them manage their time more efficiently. Our bot has two main time management commands: reminder (!reminder {time}), and timer (!timer {time}). When the user types these commands, Study Buddy will immediately start a timer and notify the user when it is over. To create an incentive to study, we also put random, funny cat memes at the end of each timer.

Study Buddy also has other fun commands to incentivize even more productivity. For example, there are hilarious motivations, roasting, quotes and even a wide variety of cat memes!

There’s still more! We noticed that students often have problems finding resources/good resources for school subjects, so we included some links for the users. Using the command !study help, users can find resources for various topics such as CS, math, art, and more.
